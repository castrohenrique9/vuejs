# VueJS com Bootstrap4 consumindo end-point

Este App utiliza Axios para consultar um end-point e mostrar Cards com o resultado.

## Entre na pasta do projeto e inicialize o container docker
```
docker-compose up
```

## Abra o App com a porta 8085
http://localhost:8085

Caso a porta 8085 já esteja sendo utilizada por outro App altere-a no arquivo docker-compose.yml
